allone.c:
в общем случае программа принимает на вход 3 параметра командной строки.
для включения режима capture:
./allone capture
для использования функции распознования прямого изображения:
./allone recognize shifted t3.pnm
для использования функции распознования повернутого изображения:
./allone recognize straight s3.pnm
