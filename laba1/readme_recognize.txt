recognize.c
для работы с программой:
расспознование прямого изображения:
./recognize straight 1.pnm
расспознование повернутого изображения:
./recognize shifted 1.pnm

полученный числовой код таблицы будет отображен на экран. для повернутого изображения так же будет выведено цифровое представление восстановленной матрицы изображения.
